# TechVest

TechVest is a simple React web application designed to help users analyze company financial data using AI insights.

## Project Overview

This is the Week 1 prototype of the TechVest interface. The app provides a basic layout and routing to show the core pages and features planned for the platform.

## Planned Features

- **AI Earnings Call Summarizer**: Summarize quarterly earnings calls.
- **AI Competitor Comparison**: Compare competitor performance.
- **Financial Ratio Analyzer**: Analyze standard financial ratios.

## Current Progress (Week 1)

During Week 1, we set up the basic project structure:
- **Navigation**: Persistent top menu bar with mobile collapse support.
- **Routing**: Configured page routing using React Router.
- **Components**: Created basic reusable card layouts.
- **Styling**: Structured basic CSS layout with simple styling.

## Tech Stack

- **React** (v18+)
- **Vite**
- **React Router DOM** (v6+)
- **Vanilla CSS**

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Piyushbhatt944/TechVest.git
   cd TechVest
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
